# OpenPaste First-Users Dashboard

Update this once per week during the first-users roadmap.

## Real user definition

An external person who installs OpenPaste and produces at least one verifiable usage signal:

- feedback issue or Discussion
- DM or email with concrete workflow notes
- call or async interview response
- second-touch reply after first install

## Activation checks

- Could the user install without help?
- Did they trigger the hotkey successfully?
- Did they paste something useful in the first session?
- Did they understand what is shipped today vs. roadmap later?

## Retention checks

- Who replied again after 7+ days?
- Which workflow made them keep the app installed?
- Which objection blocked reuse?
