### Added
• Added a new Send Feedback flow in Settings → About, with a guided form that pre-fills your OpenPaste version, macOS version, and install method before opening a draft for review.
• Added structured GitHub feedback handoff for workflow feedback, bug reports, and feature requests so you can send detailed context without retyping the basics.
• Added private Mail draft handoff for praise and other feedback, including optional quote permission when you want to share something kind. The app blushes politely.

### Fixed
• Fixed Bottom Shelf card clicks so items highlight immediately instead of feeling delayed while macOS decides whether you might double-click.
• Fixed Bottom Shelf keyboard control after selecting a card, so arrow keys and Delete go to shelf navigation more reliably without stealing input from modal sheets or text fields.
• Fixed repeated image metadata decoding in Bottom Shelf previews by caching original image dimensions, which improves responsiveness when moving between image cards.

### Changed
• Changed Bottom Shelf card activation to use a native button interaction path, preserving double-click to paste while making single-click selection feel more direct.
• Changed the About screen messaging to reflect OpenPaste’s current local-first, developer-focused positioning more clearly.
• Changed the Settings sidebar labels to improve clarity, including an explicit Sync (Premium Beta) section title.
